Hi James,

I'm writing from the Mailspring team. Thanks for including Mailspring in your TecMint roundup of Linux GUI email clients — TecMint is a trusted resource for Linux users and we're glad to be part of it.

We've shipped significant updates recently that I thought you'd want to know about for any future revisions:

**Full CalDAV calendar:** Month, week, day, and agenda views. Event creation and editing with server sync, drag-to-reschedule, recurring events with exception handling, and calendar search. Built in — no plugins required.

**Linux improvements:**
- Native Wayland support via Electron 39
- ARM64 (aarch64) builds now available
- Official AUR package
- Fixed system tray on GNOME, Unity, Ubuntu 25
- Dark/light tray icon support
- Thin window framing with hamburger menu (Linux default)
- Improved Fedora compatibility
- IPv6 SMTP support, Linux Do Not Disturb detection

**Other additions:**
- Grammar checking (LanguageTool integration)
- One-click unsubscribe (RFC 2369/8058)
- Accessibility overhaul
- VCard import/export
- Sync engine stability improvements (memory leaks, race conditions, iCloud fixes)

Our full changelog is at https://github.com/Foundry376/Mailspring/blob/master/CHANGELOG.md if you'd like to dig into the details.
